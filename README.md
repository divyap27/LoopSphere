# LoopShere

A Spring Boot web application that allows users to register and log in with encrypted passwords using BCrypt. Data is stored using JPA and a relational database.

## 🚀 Features

- 📝 User Registration
- 🔐 User Login with password validation
- 🔒 Password encryption (BCrypt)
- 💡 Basic HTML Forms (no JS)
- 🛢️ JPA Database Integration

## 🛠️ Tech Stack

- Java 17+
- Spring Boot
- Spring Data JPA
- Spring Security
- HTML, CSS
- MySQL / H2
